¿Qué es una etiqueta semántica en HTML?

Es una etiqueta que tiene un significado claro tanto para el navegador como para el desarrollador. Ejemplos de esto son <header>, <article> y <footer>.

¿Necesito un servidor web para ver mi HTML?

No. Los archivos .html se ejecutan directamente en cualquier navegador web local abriendo el archivo desde tu disco duro.
